This was a project was built demonstrate Full-stack developement skills using React for Font-end and Firebase for back-end.

The first key feature to note is the Google authentfication used to log into the web app. Once logged in, the user's username and profile image displayed using the Google Context API.

The application also allows the user to add channels which create an instance in the database.

Each channel can also accept messages that are recorded as well. The messages will display the user's username, profile image, timestamp and the message.

More product featues will be added with future developements.

Technologies:
  • ReactJS
  • Firebase
  • Google Context API
  • JavaScript
  • HTML/CSS
